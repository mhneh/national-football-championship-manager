Nhập môn Công nghệ Phần mềm
Hướng dẫn cấu hình dự án

1. Yêu cầu:
- Node > 16
- npm
- PostgresSQL
- PgAdmin 4

2. Các bước setup:
- Mở PgAdmin và import file db.sql vào PostgresSQL để tạo cơ sở dữ liệu.
- Mở thư mục source, sử dụng cmd hoặc terminal chạy lệnh: npm instal (Cài thư viện, phụ thuộc). Kết thúc, gõ tiếp: npm start.
- Mở trình duyệt, truy cập đường dẫn localhost:3000 để sử dụng ứng dụng.
